# Tasks Managment App

## Objectif du projet 

Développer une application web de gestion de tâches d’équipe permettant de manipuler les fonctionnalités principales de Laravel. 

## Fonctionnalités à implémenter 

L’application devra inclure : 

Authentification des utilisateurs (inscription / connexion)
Création et gestion de projets 
Création, modification et suppression de tâches 
Attribution d’une tâche à un utilisateur 
Ajout de commentaires sur une tâche 
Gestion du statut des tâches (à faire, en cours, terminé) 
Gestion des priorités et des dates limites 
Ajout d’étiquettes (tags) 
Upload de pièces jointes (optionnel) 


## Technologies utilisées 

- Laravel 
  Blade 
- MySQL / PostgreSQL 
- Eloquent ORM 
- Migrations & Seeders 


## Étapes du projet 

Le travail se déroulera en plusieurs étapes : 

- Installation de l’environnement Laravel 
- Création de la base de données et des migrations 
- Création des modèles et relations 
- Mise en place des contrôleurs et routes 
- Création des vues avec Blade 
- Gestion des autorisations 
- Ajout des fonctionnalités avancées (commentaires, tags, fichiers) 


## Compétences visées 

À la fin de ce projet, vous serez capables de : 

- Comprendre l’architecture MVC 
- Créer une application complète avec Laravel 
- Manipuler une base de données avec Eloquent 
- Construire un CRUD complet 
- Gérer l’authentification et les permissions
